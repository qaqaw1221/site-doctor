# Site Doctor

AI-powered платформа для автоматического поиска и исправления ошибок на сайтах

## 🚀 Быстрый старт

### Запуск бэкенда:
```bash
cd server
npm install
npm start
```
Сервер запустится на `http://localhost:3002`

### Запуск фронтенда:
```bash
cd client
python -m http.server 8000
```

## 📋 Функционал

### ✅ Реализовано:
- **Сканирование сайтов** на скорость и SEO
- **Поиск ошибок** (битые ссылки, мета-теги)
- **Базовый интерфейс** с результатами
- **Сохранение результатов** в MongoDB

### 🔄 В разработке:
- **Авто-исправления** простых проблем
- **AI-анализ** контента
- **Система подписок** для монетизации
